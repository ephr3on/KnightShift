# Match Settings Simplified

Updated the Online Lobby host settings to be more mobile-friendly and easier to understand.

## What changed

- Replaced the crowded `Match Settings (Host)` panel with a simpler `Choose the match vibe` section.
- Added three one-tap presets:
  - Quick Duel
  - Balanced
  - Pro Match
- Kept quick controls visible only for the most common choices:
  - Difficulty
  - Board
  - Rounds
- Moved less common controls into a collapsible `Advanced rules` panel:
  - Turn rule
  - Timer
  - Move cap
- Added a compact summary strip so both host and guest can instantly understand the current setup.
- Redesigned the guest view so it appears as a clean read-only summary instead of a disabled complex settings panel.

## Build

`npm run build` passed successfully.

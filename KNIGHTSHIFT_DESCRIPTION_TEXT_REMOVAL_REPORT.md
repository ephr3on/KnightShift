# KnightShift Description Text Removal

Removed non-essential description copy from the main UI surfaces to create a cleaner, faster mobile-first interface.

## Updated areas
- Main menu hero and mode cards
- Campaign journey hero, zones, and footer copy
- Online menu invite/create/join helper copy
- Online lobby invite/settings/ready helper copy
- Screen headers no longer display subtitles
- Credits reduced to a compact title-only panel
- Daily, archive, saved, and custom puzzle headers simplified

## Preserved
- Button labels
- Essential gameplay labels
- Errors and copy feedback
- Scores, progress, move counts, and puzzle metadata

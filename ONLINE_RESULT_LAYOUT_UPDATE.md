# Online Result Layout Update

Updated the online result screen to match the requested compact layout:

- The two player result cards are shown next to each other.
- The puzzle stats and settings cards are shown next to each other.
- The first two action buttons are shown side by side.
- The lower two action buttons are shown side by side.
- Mobile sizing was tightened so the result screen is less tall and easier to scan.

Build checked with `npm run build`.
